# Contributing

Thank you for improving the SMSFlow Python SDK.

Before submitting changes:

- Do not include real SMSFlow credentials, bearer tokens, phone numbers, logs, or customer data.
- Keep the public API simple and easy to use from backend applications.
- Add or update tests for behavior changes.
- Run `python -m unittest discover -s tests` before opening a pull request.

Documentation: https://docs.smsflow.co.za/
